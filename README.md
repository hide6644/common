## COMMON
[AppFuse][]を参考に、技術検証のため独自に修正を加えている。

## DESCRIPTION
単純なWebアプリケーション。  
データベースアクセス、ログイン認証などの基本的な機能を含んでいる。

また、実装には複数のオープンソースを使用している。代表的なものは以下の通り。
* Bootswatch
* Freemarker
* Hibernate
* Hibernate Search
* Hibernate Validator
* HikariCP
* Jxls
* Spring
* Spring Security
* Thymeleaf
